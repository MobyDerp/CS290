Welcome!

This how-to guide will show you how to be able to leverage the Bureau Labor of Statistics(BLS) data using their Application Program Interface (API).

Some Prerequiste

This guide assumes you have a basic knowledge of computers, HTML/CSS, JavaScript, server side programming, and JavaScript Object Notation (JSON).

Learning Objective of How-to-API

  • Understand what is an API
  • Understand how to access Series ID format for data pull
  • Learn how to run Single Series
  • Learn how to run Multiple Series
  • Learn how to run One or More Series with Optional Parameters
